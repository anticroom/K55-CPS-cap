# K55 CPS Cap

## What is this?
This is a recreation of the K55 CPS cap, recreating the hardware limitations found in the **Corsair K55/70** keyboards which caused the CPS to max out at 16.66666... allowing perfect wave consistency spam at that cps.

This also allows you to change the CPS cap to anything which may match other known input devices with similar hardware limitations if needed.
